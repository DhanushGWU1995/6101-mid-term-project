Understanding the CHD Risk Prediction Model

This prediction is based on a classification tree model trained on BRFSS 2024 data with the following key factors:

  • Age Group (18-69 years)
  • Sex (Male/Female)
  • Smoking Status
  • Diabetes Status
  • Exercise in Past 30 Days
  • General Health Status

Model Performance Metrics

The decision tree was trained using balanced priors to handle class imbalance.
